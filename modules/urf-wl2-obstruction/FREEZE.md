# FREEZE POLICY

## Canonical Surface

The following files define the repository claim boundary:

- README.md
- STATUS.md
- CITATION.cff

## Supporting Surface

The following are non-canonical support artifacts unless explicitly promoted:

- scripts
- workflows
- registries
- certificates
- notes
- generated artifacts
- local environments

## Mutation Policy

- `main` tracks the canonical repository identity.
- No change may reintroduce unrelated repository framing.
